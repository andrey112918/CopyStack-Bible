# CopyStack Vocabulary

This document defines the official terminology used across the CopyStack ecosystem.

---

# AI Employee

An autonomous AI system capable of performing a specific business role.

Example:

- Video Employee
- Research Employee
- Outreach Employee

---

# Engine

A reusable software module responsible for a specific capability.

Examples:

- Video Engine
- Website Engine
- Automation Engine

---

# CopyStack Core

The orchestration layer responsible for coordinating all AI Employees and system components.

---

# Blueprint

A structured description of an AI Employee.

It defines:

- Inputs
- Outputs
- Memory
- Workflow
- Integrations
- Rules

Blueprints are transformed into AI Employees.

---

# Memory

Information permanently stored by CopyStack.

Types:

- User Memory
- Company Memory
- Agent Memory

---

# Rulebook

A collection of editing or behavioral rules followed by an AI Employee.

---

# Workflow

The ordered sequence of actions executed by an AI Employee.

---

# Agent Factory

The system responsible for generating new AI Employees from Blueprints.

---

# Human-in-the-Loop

A workflow where a human reviews or improves AI-generated output before delivery.

---

# SSOT

Single Source of Truth.

The CopyStack-Bible repository containing the official documentation of the company.
